# pig-game

#THE PIG GAME (project 1)

#Description

This project is a simple dice game created using python. 2-4 players take turns rolling
a dice to earn points. The goal is to be the first player to reach the winning score.

##How to play 


*players choose to roll or hold during their turn.



*rolling adds points unless a 1 is rolled.

*rolling a 1 ends the turnn and removes turn points.


*holding saves points and ends the turn.


*first player nto reach the winning score wins 

##Requirements


python 3

1. open the project file.
2. run the python script.
